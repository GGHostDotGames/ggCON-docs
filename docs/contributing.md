# Contributing

## Report a bug

Found a bug or issue with ggCON? Open a [Bug Report](https://github.com/GGHostDotGames/ggCON-docs/issues/new?template=bug_report.yml){target=_blank} on the ggCON-docs repository.

## Request a feature

Have an idea for ggCON? Open a [Feature Request](https://github.com/GGHostDotGames/ggCON-docs/issues/new?template=feature_request.yml){target=_blank} on the ggCON-docs repository.
