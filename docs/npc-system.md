# NPC System

!!! info "Not yet available"
    The AI NPCs plugin is in development and **not yet available**. When it ships, it will let you create characters that interact with players through in-game chat. This page will be updated with setup steps then.

**Looking for NPC tracking or kill events today?** Those are available now via two published plugins:

- **[NPC Tracker](plugins.md#npc-tracker)** — live tracking of guards, drifters, zombies, animals, sentries, drones, and Razor, with health bars, alive/dead status, an NPC kill log, and map markers
- **[Kill Feed](plugins.md#kill-feed)** — a real-time feed of PvP, NPC, Animal, Trap, and Suicide kills, with a leaderboard, Discord webhooks, and in-game broadcasts

When the AI NPCs plugin is released, it will add an NPCs tab to the web panel for managing the system and choosing an AI model. Until then, there is no NPC management UI in ggCON.
